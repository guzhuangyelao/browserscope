Async Attribute for Scripts







This test determines if your browser supports the HTML5 async attribute for scripts. This page uses four scripts with the following source order:

  1. async external script (takes 4 seconds to download)
  2. async external script (takes 2 seconds to download)
  3. inline script
  4. normal external script (downloads immediately)

There is also an onload handler script that executes when the page and all its resources have finished downloading.

If the async attribute works, the scripts will have the following expected execution order:

  1. inline script
  2. normal external script
  3. async script (2 second download)
  4. async script (4 second download)
  5. onload handler
